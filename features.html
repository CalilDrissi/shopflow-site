Triggers

Shopify API

AI Nodes

Generate text, translate content, analyze sentiment. Bring your own API key — works with OpenAI, Anthropic, and custom endpoints.

Logic & Flow

Filter, sort, loop, branch, and merge data. Build conditional paths and complex multi-step logic without writing code.

Outputs

Canvas Editor

Drag. Drop. Connect.

The visual canvas editor makes building workflows intuitive. Add nodes from the palette, drag them into position, and connect them with a click. Configure each node's settings in the side panel.

  • Infinite canvas with zoom and pan
  • Color-coded node categories
  • Real-time node count and connection stats
  • Export and import workflows as JSON

Scheduling

Set it and forget it

Schedule workflows to run automatically. Choose from hourly, daily, weekly, or custom intervals. Built on reliable cron with execution timeouts and failure handling.

  • Interval: every N minutes/hours
  • Daily at a specific time
  • Weekly on a specific day
  • 5-minute execution timeout safety net

Run Logs

Full visibility into every execution

Every workflow run is logged with node-by-node results, timing, items processed, and error details. Debug issues instantly and track your automation's performance over time.

  • Node-level execution results
  • Items processed count per run
  • Error details with stack traces
  • Run history with status indicators
