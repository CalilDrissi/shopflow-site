Journey

How we got here

Early 2026

Desktop App Launch

Started as a Tauri desktop app with a Rust backend. 46 node types, visual canvas editor, local workflow execution. Proved the concept worked.

March 2026

Web SaaS Conversion

Rebuilt the backend in Node.js + PostgreSQL for multi-tenant SaaS. Added auth, billing, scheduling, and email verification. Deployed at app.shopifyer.link.

March 2026

Dropshipping Templates

Added 15 dropshipping workflow templates covering 10 major supplier platforms — AliExpress, CJ, Spocket, Printful, DSers, and more.

Now

Growing Every Week

New templates, new integrations, admin dashboard, and security hardening. Building in public, shipping fast.

Under the Hood

Built with modern tools

Node.js + Express

Fast, reliable backend with JWT auth, rate limiting, and Helmet security headers.

PostgreSQL

JSONB storage for workflows and node configs. Full ACID compliance for your data.

Vanilla JS + Vite

Zero-framework frontend. Fast, small, no build complexity. Just HTML, CSS, and JS.

Stripe Billing

Secure checkout, customer portal, webhook-driven subscription management.

Shopify Admin API

Full access to products, orders, customers, inventory, collections, and metafields.

Any AI Provider

OpenAI-compatible API support. Bring your own key — works with GPT, Claude, local models.
